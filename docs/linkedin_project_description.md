# LinkedIn Project Description

I completed an end-to-end Supply Chain Delivery Performance Analysis portfolio project using Python, pandas, and SQLite. The project starts from a local raw ZIP archive, profiles and cleans 180,519 order-item records, defines grain-aware KPIs, validates Python metrics against SQL, documents a Power BI semantic model, and delivers a finished interactive HTML dashboard.

The analysis focuses on late delivery performance, shipping mode effectiveness, regional operational risk, profitability, and high-sales low-margin segments. It is structured as a practical business analytics case study rather than a generic notebook.
