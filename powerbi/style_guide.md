# Dashboard Visual Style Guide

## Canvas
Use 16:9 widescreen. Keep a consistent 24 px outer margin and align visuals to a simple grid.

## Typography
Use Segoe UI or Aptos. Page title 22-26 pt, section headers 12-14 pt, labels 9-11 pt.

## Palette
- Primary: muted blue `#2F5F8F`
- Positive performance: green `#2E7D32`
- Risk: red `#B23A48`
- Warning: amber `#C77D00`
- Neutral text: charcoal `#2B2B2B`
- Background: off-white `#F7F8FA`

## Formatting
- Currency: show in millions with one decimal where useful.
- Percentages: one decimal place.
- Days: one decimal place.
- Use conditional formatting sparingly for risk and margin.

## Accessibility
Use high contrast between text and background. Avoid relying on color alone; include labels and tooltips for key risk visuals.
